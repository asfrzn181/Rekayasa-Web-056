Pada semester ini mahasiswa mempelajari tentang WordPress. Dengan menyiapkan XAMPP, Browser dan WordPressnya yang bisa di download disini

Berikut tata cara penginstalannya.

  1. Extract folder WordPress ke C:\xampp\htdocs dengan nama folder bebas misalkan wordpress056
  2. Lalu buka browser, masukkan localhost/<nama folder wordpress> misalkan localhost/wordpress056/
  3. Nanti akan tampil seperti ini, silahkan pilih bahasa yang anda mengerti

4. Lalu buatlah database di localhost/phpmyadmin dengan nama db_<nama> misalkan db_wordpress056 seperti saya gunakan. Hanya buatkan saja tanpa menambahkan table apapun

5. Lalu lanjutkan install wordpressnya

Database Name : db_wordpress056
username : root
Password : <kosongkan>
Database Host : localhost
Table Prefix : wp_

6. Tinggal kita Submit dan Run install. Maka WordPress pertama kita telah selesai
