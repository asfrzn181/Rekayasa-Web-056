Tutorial Install Tema

  1. CARA PERTAMA
  • masuk ke bagian Dashboard WordPress, maka silahkan untuk memilih menu appearance yang tersedia dalam halaman awal. Selanjutnya klik Themes dan halaman Themes pun akan terbuka secara otomatis.
  • Kemudian untuk mengganti tema atau template, Anda hanya perlu memilih menu Add New.
  • Pastikan terhubung ke internet terlebih dahulu, lalu anda dapat memilih tema yang sesuai dengan wordpress anda.
  • Jika dipastikan sudah cocok, anda dapat mengklik install
  • Tunggu beberapa saat proses installnya, setelah itu anda dapat mengklik Active untuk mengaktifkan Tema atau Preview untuk melihat tema

2. CARA KEDUA

  • Buka Google atau browser lainnya, masukkan keyword pada search engine Download Tema WordPress Gratis yang sesuai dengan selera anda atau click me!
  • Download Tema yang telah anda pilih. Yang biasanya file tersebut akan berformat .zip
  • Jika download sudah selesai, masuk ke dalam dashboard. Pilih Appearance > Theme > Add New > Upload Theme > Pilih Thema yang baru didownload tadinya > install now. Tunggu beberapa saat
  • Jika sudah kembali ke menu daftar tema. Maka akan muncul Thema yang baru diinstall tadinya
  • Klik Active pada Thema untuk mengaktifkannya atau Preview untuk melihat – lihat saja
  • Selesai

3. CARA KETIGA

Caranya mirip – mirip dengan cara yang kedua namun berbeda disini dengan cara manual dari direktori

  • Pastikan sudah ada tema yang ingin di install.
  • Buka Tema tersebut dengan Winrar atau tools extract file lainnya
  • masuk ke C:\xampp\htdocs\wordpress056\wp-content\themes <tergantung didirectory mana XAMPP anda tersimpan>.
  • Extract file tema tersebut ke dalam direktori tersebut
  • setelah selesai proses perextrackaan.
  • Anda dapat membuka Dashboard wordpress > Appearance > Theme. maka akan otomatis muncul tema yang anda extract tadi.
