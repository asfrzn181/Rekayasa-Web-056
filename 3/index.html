Tatacara memposting Pos di wordpress. Disini saya akan menggunakan 2 role, yaitu kontributor untuk menulis artikel dan editor untuk mengreview serta acc postingan.

Kontributor

  1. Login. Kedalam WordPress

2. Pada menu, Klik Post > Add New

3. Dapat diisi sesuai keterangan, Artikel suka suka

4. Jika ingin mengedit font, tinggal blok kata – kata yang ingin diedit. Maka akan muncul menu seperti berikut. Jika tidak ada di menu, kita dapat mengedit dalam bentuk HTML hanya dengan mengklik titik tiga lalu Edit as HTML

5. Dapat juga mengubah ukuran Font atau menggunakan DROP CAP dengan memilih pilihan disamping kanan.

6. Selanjutnya menyisipkan gambar atau video, dapat dilakukan dengan cara mengklik tanda tambah Block (hitam) atau Tambah biru pada sudut kiri atas.

7. Dalam menambahkan gambar dapat menggunakan URL atau tinggal Drag gambar dari Explorer. Untuk menyisipkan file atau gambar. user kontributor tidak dapat mengupload langsung atau menggunakan cara drag file. Tetapi untuk tingkat atas seperti Editor dapat melakukannya. Oleh karena itu saya hanya menggunakan URL saja.

8. Cara mengambil url gambar, hanya dengan klik Copy Image Address. Lalu pastekan pada Insert from URL maka akan muncul gambar tanpa harus mendownload terlebih dahulu

9. Setelah selesai semuanya, Tinggal Publish dan Submit for Review

Editor

  1. Login sebagai Editor
  2. Masuk ke All Post – Masuk ke Artikel yang ingin diedit atau di Review. Jika sudah sesuai dapat dipublish

3. Jika sudah terpublish, maka akan langsung muncul ke halaman utama

4. Kita akan mengkategorikan Postingan artikel agar tersedia pada tempatnya

5. Setelah sudah dibuat maka akan muncul ke tabel jenis kategori

6. Buat artikel baru atau buka artikel yang belum terkategori, lalu ganti Categories pada Sub Menu Post settingan artikel

7. Jika berhasil maka akan muncul pada menu halaman utama

8. Dan inilah hasilnya
