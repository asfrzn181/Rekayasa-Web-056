Pada pertemuan kedua dosen pengampu menjelaskan tentang menu menu pada wordpress

A. Cara Login ke wordpress

  1. Buka Browser masukan url localhost/wordpress056/wp-login.php maka akan muncul tampilan

2. Tinggal masukkan username dan password yang telah terdaftar sebagai admin atau user pada database yang telah dibuat sebelumnya

B. Ubah Profil dan Password

  1. Pilih User > Profile pada Dashboard

2. Tinggal ubah – ubah isinya sesuai ketentuan

C. Daftar User baru

  1. Buka menu user >tambahkan user baru

2. Masukkan data sesuai yang dibutuhkan, pilihlah role sesuai kebutuhan user. Lalu submit
